Predictive Analytics & Alerts on Privileged Email Communication

A tool that scans all outgoing email communication and emails from a domain or set of domains that mention clients that are indicated as privileged.

Tool learns from flagged emails and can predictively flag potential issues w/out the need for admin. listing of specific names.

Emails Flagged Source on GitHub
This project is a Fintech Hackathon sponsered by RBC Capital Markets, check out the Zahn Innovation Center Here.

How Does This Email Flag App Works

  • This app was deployed to Heroku, using Git and can be found Here
  • We assume that we are receiving emails from a domain to parse and check whether or not they will be flagged based on specific information
  • After cleaning up the source code for the emails, we parse through the emails to look for potential flags
  • After flagging the emails based on email address, names and the content or subject of the email we extract the flagged and not flagged emails into a seperate folder
  • The Pages in the navigation bars will show the names of the flagged emails and details also the machine learning progress
